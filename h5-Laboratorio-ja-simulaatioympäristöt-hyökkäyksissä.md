# h5 Laboratorio- ja simulaatioympäristöt hyökkäyksissä  

a) ARP hyökkäys  
Aloin tunnilla tekemään ARP hyökkäys tehtävää. Otin Windows koneellani ssh yhteyden mininet virtuaalikoneeseen. Ajoin komennon `ryu-manager ryu.app.simple_switch_13`. Tämän jälken otin PowerShellillä ssh yhteyden mininet virtuaalikoneeseen. Tämän jälkeen kokeilin Larin avulla komentoja `xauth list $DISPLAYS`, `xauth list $DISPLAY` ja `sudo xauth list $DISPLAY`, mutta sain kokoajan saman ongelman.
